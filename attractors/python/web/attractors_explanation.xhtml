Strange attractors ?
A couple of years back I stumbled on Julien Sprott's book on polynomial strange attractors.

It gives a nice set of algorithms to check for attractor convergence. The explanations of the theory are crystal clear.

The programming examples are a bit dated, but I had some spare time by then, so I just rewrote the algorithm in Python. Pretty soon, tons of nice pictures started to pile up in my hard drive.

The code sat there for a couple of years until lately, where I did have more spare time and decided to refine the coloring part a bit, and write a simple daily generator.
